# 作业 1

根据以下要求创建网页。你应该使用 Notepad 或类似的基于文本的编辑器编写所有 HTML 和 CSS 代码。**不允许**使用任何 AI、Web 开发软件或前端开发框架（例如 vue）。

**详细要求**

1.  **第一页必须作为小组/团队页面，并包含以下必需内容：**
    *   团队成员的姓名和学号。
    *   指向每位成员个人主页的超链接。
    *   关于**合肥**或**苏州**的介绍：
        *   该部分必须包含涵盖地理位置、人口统计、风景、文化、美食等的描述性文字。
        *   包含展示城市美景的图片。
        *   包含用于获取更多信息的外部链接（部分链接必须附加在图片上，点击时激活）。
    *   **必须**使用自定义的**嵌入式 CSS 样式 (Embedded CSS)** 来美化页面，需满足以下最低标准：
        *   至少一个针对两个或更多 HTML 标签的**共享选择器 (Shared selector)**。
        *   至少一个**基于类的选择器 (Class-based selector)**。
        *   至少一个**上下文样式 (Contextual style)**（应用于嵌套在另一个标签内的标签的样式）。
        *   `<a>` 标签的**悬停效果 (Hover effect)**。
        *   所有定义的样式必须实际应用到页面内容中。
        *   多个**内联 CSS 样式 (Inline CSS)**。
    *   实施**响应式设计**以确保跨设备的兼容性。

2.  **每位团队成员必须开发两个个人页面：一个主页 (Homepage) 和一个职业技能页面 (Career Skill Page)。**

    **主页 (Homepage)** 必须侧重于个人介绍，并包括：
    *   你的照片、全名和学号。
    *   通过文本、图片和/或视频片段呈现的个人信息（例如背景、爱好、兴趣）。
    *   指向你的电子邮件地址以及另外两个页面（团队页面 + 职业技能页面）的超链接。
    *   清晰的职业技能发展计划大纲。

    **主页的样式规则：**
    a) **外部 CSS 样式表 (External CSS)**（强制要求）必须包括：
    *   格式化所有 `<p>`（段落）标签的样式。
    *   一个通用类定义（在页面元素中复用）。
    *   一个特定于标签的类（例如，`.header h1` 用于设置头部内的标题样式）。
    *   带有悬停效果的 `<a>` 标签类（例如，`.link:hover`）。

    b) **嵌入式 CSS (Embedded CSS)**（在页面的 `<style>` 标签内）必须包括：
    *   用于字体样式的 ID 选择器（例如，`#bio-text { font-family: Arial; }`）。
    *   文本标题的样式（例如，`h2 { color: #333; }`）。
    *   上下文样式（例如，`div.content p { line-height: 1.6; }`）。

    必须实施**响应式设计**，以确保与移动设备、平板电脑和桌面设备的兼容性。

    **职业技能页面 (Career Skill Page)** 必须：
    *   清楚地列出实现职业目标所需的**硬技能**（技术/专业能力，如编程语言、数据分析）和**软技能**（人际/可迁移技能，如沟通、解决问题）。
    *   应用多个**内联 CSS 样式 (Inline CSS)**（直接在 HTML 元素内）来格式化内容（例如，文本颜色、间距、字体大小）。

**提交**

将所有网页、CSS 样式文件和其他相关文件保存在一个文件夹中，并将该文件夹压缩为 zip 文件。Zip 文件名必须包含团队编号和学号。
